# Security policy

Report vulnerabilities through GitHub's private
[security advisory form](https://github.com/pawnkit/pawnlint/security/advisories/new).
Please do not open a public issue before a fix or mitigation is available.

Pawn source, configuration, baselines, and external rule output are untrusted.
Crashes, hangs, path escapes, unsafe fixes, and excessive resource use are in
scope.
